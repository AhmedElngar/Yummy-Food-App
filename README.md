## Overview
Yummy is a modern, user-friendly recipe finder built using **HTML**, **CSS**, **JavaScript**, and an external **Recipe API**.  
Users can explore meals by category, ingredients, or origin, and view detailed cooking instructions.

## Features
- Search recipes by name or ingredient  
- Filter meals by category or area  
- View recipe details, ingredients, and steps  
- Responsive design (mobile-friendly)  

## Technologies Used
- HTML5 / CSS3 / JavaScript (ES6)  
- Fetch API  
- Bootstrap 5
